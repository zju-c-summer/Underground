# 总体目标

本项目主要实现一个迷宫探险的游戏，能够生成一个适合大小
的迷宫，具有一个图形化的界面，玩家可以在其中进行自由探险，
并以走出迷宫为最终目标。

# 系统需求

软件开发和运行在Windows环境下，在各版本的Windows下均可运行。
主要以Visual Studio 2017 为开发环境，使用的工具包括CMake，Qt等。

# 功能分析

1. 开始界面

	玩家点击开始进行游戏。

	
2. 游戏画面

	地图显示以玩家为中心一定视野范围内的平面物体，人物基本处于画面中央。
	可以通过键盘WASD来对人物进行控制，可以上下左右进行移动。
	初步预期只有墙壁和道路，地图有一个起点和一个终点，到达重点游戏成功。
	之后可能会加入怪物、其他物品和战斗系统等。

3. 小地图

	小地图显示已经被玩家探索过的地图形状。
	小地图显示终点相对于玩家的方位。（暂定）
